# 📘 Assignment-8: Flexbox Layout Design
📌 Project Overview

This project is a responsive layout design created using HTML5 and CSS3 (Flexbox).
It demonstrates structured page layout with:

- Header section

- Sidebar (Navbar – 30%)

- Content area (70%)

- Nested flex rows and columns

- Colored bordered boxes (Red, Blue, Green) for layout visualization

## 🛠 Technologies Used

- HTML5

- CSS3

- Flexbox
